# Web 云管理面板指南

[English](./en/cloud.md)

R-Bot 客户端内置了完整的 Web 云管理面板，在浏览器中即可管理 Oracle Cloud、Azure、SolusVM、Cloudflare DNS 等多云平台资源，操作能力与 Telegram 机器人对齐。

---

## 访问方式

启动客户端后，访问：

```
https://你的IP:9527
```

登录后点击左侧导航栏进入云管理模块。

> 首次使用需先在「设置」中配置云平台 Profile（或通过 `client_config` 文件配置）。

---

## 概览面板

登录后的首页展示当前 Profile 的概览信息：

| 卡片 | 内容 |
|------|------|
| **成本** | 近 3 个月花费明细 |
| **流量** | 近 3 个月流量使用量 |
| **订阅** | 订阅类型、支付方式、有效期 |
| **配额** | 实例、网络、存储配额使用情况 |

支持切换不同的 Profile 查看不同账户的信息。

---

## Oracle Cloud 管理

### 实例管理

| 操作 | 说明 |
|------|------|
| 查看实例列表 | 展示所有实例及状态 |
| 启动 / 关机 / 重启 | 基础实例操作 |
| 终止实例 | 删除实例（可选保留引导卷） |
| 重装系统 | 重置为初始系统镜像 |
| 升降配 | 调整 CPU 和内存规格 |
| 修改名称 | 修改实例显示名称 |
| 修复实例 | 尝试修复异常实例 |

### 网络 / IP 管理

| 操作 | 说明 |
|------|------|
| 查看 VNIC 列表 | 展示网卡、私有 IP、公网 IP |
| 换 IPv4 | 更换公网 IPv4 地址 |
| 换 IPv6 | 更换 IPv6 地址 |
| 删除 IP | 删除公网 IP 或 IPv6 |
| 附加 IPv4 | 给实例添加额外的 IPv4 |
| 附加 IPv6 | 创建新的 IPv6 地址 |
| 附加保留 IP | 附加预留的公网 IP |

### 卷管理

| 操作 | 说明 |
|------|------|
| 查看卷列表 | 展示所有块存储卷 |
| 调整容量 | 增加卷大小（不可缩小） |
| 调整 VPU | 修改性能单位提升 IO 速度 |
| 批量 VPU | 一键拉满所有卷的 VPU |
| 分离卷 | 将卷从实例分离 |
| 删除卷 | 永久删除块存储卷 |

### 用户管理

| 操作 | 说明 |
|------|------|
| 查看用户列表 | 展示租户下所有用户 |
| 创建用户 | 添加新的管理员用户 |
| 修改邮箱 | 更新用户邮箱地址 |
| 重置密码 | 重置用户登录密码 |
| 清除 2FA | 清除用户绑定的两步验证 |
| 修改租户名 | 更改租户显示名称 |
| 删除用户 | 删除指定用户 |

### 对象存储 (OSS)

| 操作 | 说明 |
|------|------|
| 浏览 Bucket | 列出所有存储桶 |
| 浏览文件 | 支持前缀过滤和分页浏览 |
| 上传文件 | 流式上传，防路径遍历 |
| 下载文件 | 流式下载 |
| 删除文件 | 删除单个对象 |
| 创建文件夹 | 创建目录结构 |

### Profile 管理

| 操作 | 说明 |
|------|------|
| 查看 Profile 列表 | 展示所有配置的 OCI Profile |
| 切换 Profile | 切换到不同的云账户操作 |
| 删除 Profile | 删除不需要的 Profile |

## Cloudflare DNS 管理

| 操作 | 说明 |
|------|------|
| 查看域名列表 | 列出 Cloudflare 托管的所有域名 |
| 查看 DNS 记录 | 查看域名下的所有解析记录 |
| 添加记录 | 新增 A/AAAA/CNAME 等 DNS 记录 |
| 修改记录 | 编辑现有 DNS 记录 |
| 删除记录 | 删除指定 DNS 记录 |

---

## Azure 管理

| 操作 | 说明 |
|------|------|
| 查看 VM 列表 | 展示所有虚拟机及状态 |
| 创建 VM | 新建虚拟机 |
| 删除 VM | 删除虚拟机 |
| 重启 VM | 重启虚拟机 |
| 换 IP | 更换公网 IP 地址 |
| 资源用量 | 查看配额使用情况 |

---

## SolusVM 管理

| 操作 | 说明 |
|------|------|
| 查看节点列表 | 列出所有 VPS 节点 |
| 节点仪表盘 | 查看 VPS 状态详情 |
| 开机 / 关机 / 重启 | VPS 电源操作 |

## 设置

### 实例监控

| 功能 | 说明 |
|------|------|
| 实例监控通知 | 定时监控实例状态，异常时 Telegram 通知 |
| 监控自启 | 异常时自动尝试启动实例 |
| 每日报告 | 定时推送花费和流量报告 |
| 一键测活 | 批量检测所有 Profile 账户状态 |

### ACME 证书

在设置页面配置 Let's Encrypt 自动证书，详见 [Web SSH 终端指南 — SSL 证书](./webssh.md#ssl-证书配置)。

## 主题

Web 界面支持多种主题切换：

| 主题 | 说明 |
|------|------|
| 经典 | 默认深色主题 |
| 其他主题 | 樱花、星空、深渊等 |

在界面右上角的主题选择器中切换。

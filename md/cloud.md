# Web 云管理面板指南

[English](./en/cloud.md)

R-Bot 客户端内置了完整的 Web 云管理面板，在浏览器中即可管理 Oracle Cloud、Azure、SolusVM、Cloudflare DNS 等多云平台资源，操作能力与 Telegram 机器人对齐。

---

## 访问方式

启动客户端后，访问：

```
https://你的IP:9527
```

登录后点击左侧导航栏进入云管理模块。

> 首次使用需先在「设置」中配置云平台 Profile（或通过 `client_config` 文件配置）。

---

## 概览面板

登录后的首页展示当前 Profile 的概览信息：

| 卡片 | 内容 |
|------|------|
| **成本** | 近 3 个月花费明细 |
| **流量** | 近 3 个月流量使用量 |
| **订阅** | 订阅类型、支付方式、有效期 |
| **配额** | 实例、网络、存储配额使用情况 |

支持切换不同的 Profile 查看不同账户的信息。

## Oracle Cloud 管理

### 实例管理

| 操作 | 说明 |
|------|------|
| 查看实例列表 | 展示所有实例及状态 |
| 创建实例 | 选择配置规格、镜像、网络等参数创建新实例 |
| 快捷开机 | 基于已保存的配置快速创建实例 |
| 启动 / 关机 / 重启 | 基础实例操作 |
| 终止实例 | 删除实例（可选保留引导卷） |
| 重装系统 | 重置为初始系统镜像 |
| 升降配 | 调整 CPU 和内存规格 |
| 修改名称 | 修改实例显示名称 |
| 修复实例 | 尝试修复异常实例 |

![创建实例](../screenshots/cloud-create.jpg)

### 网络 / IP 管理

| 操作 | 说明 |
|------|------|
| 查看 VNIC 列表 | 展示网卡、私有 IP、公网 IP |
| 换 IPv4 | 更换公网 IPv4 地址 |
| 换 IPv6 | 更换 IPv6 地址 |
| 删除 IP | 删除公网 IP 或 IPv6 |
| 附加 IPv4 | 给实例添加额外的 IPv4 |
| 附加 IPv6 | 创建新的 IPv6 地址 |
| 附加保留 IP | 附加预留的公网 IP |

### 卷管理

| 操作 | 说明 |
|------|------|
| 查看卷列表 | 展示所有块存储卷 |
| 调整容量 | 增加卷大小（不可缩小） |
| 调整 VPU | 修改性能单位提升 IO 速度 |
| 批量 VPU | 一键拉满所有卷的 VPU |
| 分离卷 | 将卷从实例分离 |
| 删除卷 | 永久删除块存储卷 |

### 用户管理

| 操作 | 说明 |
|------|------|
| 查看用户列表 | 展示租户下所有用户 |
| 创建用户 | 添加新的管理员用户 |
| 修改邮箱 | 更新用户邮箱地址 |
| 重置密码 | 重置用户登录密码 |
| 清除 2FA | 清除用户绑定的两步验证 |
| 修改租户名 | 更改租户显示名称 |
| 删除用户 | 删除指定用户 |

### 对象存储 (OSS)

| 操作 | 说明 |
|------|------|
| 浏览 Bucket | 列出所有存储桶 |
| 浏览文件 | 支持前缀过滤和分页浏览 |
| 上传文件 | 流式上传，防路径遍历 |
| 下载文件 | 流式下载 |
| 删除文件 | 删除单个对象 |
| 创建文件夹 | 创建目录结构 |

### Profile 管理

| 操作 | 说明 |
|------|------|
| 查看 Profile 列表 | 展示所有配置的 OCI Profile |
| 切换 Profile | 切换到不同的云账户操作 |
| 删除 Profile | 删除不需要的 Profile |

### Email Delivery（邮件服务）

| 操作 | 说明 |
|------|------|
| 一键搭建 | 输入域名和发件人地址，自动完成邮件域创建、DKIM 签名、DNS 记录配置、发件人注册、SMTP 凭证生成全流程 |
| DNS 模式 | 支持 Cloudflare 自动配置和手动配置两种模式，手动模式会暂停展示 DNS 记录等待用户确认 |
| 搭建进度 | 异步执行，实时进度条展示当前步骤，离开页面可恢复 |
| 域名管理 | 查看邮件域列表、DKIM 验证状态、域生命周期状态 |
| DKIM 信息 | 查看域的 DKIM CNAME 记录详情，供手动配置 DNS 参考 |
| DKIM 修复 | DKIM 验证失败时，通过 Cloudflare 自动重新配置 DNS CNAME 记录（异步，支持进度查询） |
| 发件人管理 | 查看指定域下的发件人列表和状态 |
| 添加发件人 | 为已有邮件域添加新的发件人地址 |
| SMTP 配置 | 查看 SMTP 连接信息（服务器/端口/用户名） |
| 重新生成凭证 | 重新生成 SMTP 密码（一次性显示，请妥善保存） |
| 测试发信 | 通过 OCI HTTP API 发送测试邮件，支持自定义收件人/主题/内容 |
| 删除域 | 级联删除邮件域及关联的 DKIM、发件人资源 |

## 设置

### 实例监控

| 功能 | 说明 |
|------|------|
| 实例监控通知 | 定时监控实例状态，异常时 Telegram 通知 |

## 主题

Web 界面支持 8 种主题，每种均包含亮色和暗色两种模式：

| 主题 | 风格 |
|------|------|
| 经典 | 默认蓝色，简洁商务 |
| 樱花 | 粉色系，二次元 / ACG 风 |
| 赛博 | 霓虹青蓝，科技感 |
| 水墨 | 金色古朴，国风 |
| 极光 | 青绿紫渐变，北极光 |
| 星夜 | 深紫色调，星空 |
| 海渊 | 深海蓝绿，荧光点缀 |
| 日落 | 暖橘珊瑚色，温馨 |

在界面右上角的主题选择器中切换主题，点击明暗模式按钮切换亮色/暗色。
